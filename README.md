# Todo List

A dynamic Todo List application that allows users to organize tasks into projects. Built with vanilla JavaScript using ES6 modules and Webpack.

![Demo for my Todo List](todoListDemo.gif)
## Features

- Create and manage multiple projects
- Add todos with title, description, due date, time, and priority
- Mark todos as complete/incomplete
- Clean, organized interface with sidebar navigation

## Technologies Used

- JavaScript (ES6 Modules)
- Webpack (bundling and development server)
- HTML5 & CSS3
- date-fns (date formatting)

## Getting Started

### Prerequisites

- Node.js installed on your machine

### Installation

1. Clone the repository
   ```bash
   git clone https://github.com/inshaal81/Todo-List.git
   ```

2. Navigate to the project directory
   ```bash
   cd Todo-List
   ```

3. Install dependencies
   ```bash
   npm install
   ```

4. Start the development server
   ```bash
   npm start
   ```

5. Build for production
   ```bash
   npm run build
   ```
